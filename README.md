# homebrew-jira-cli

Homebrew tap for [`jira-cli`](https://github.com/zhiyue/jira-cli) — an agent-first CLI for legacy Jira Server 8.13.5.

## Install

```bash
brew tap zhiyue/jira-cli
brew install jira-cli

# verify
jira-cli --version
```

## Updates

`Formula/jira-cli.rb` is auto-generated by the release workflow in
`github.com/zhiyue/jira-cli` on every `v*.*.*` tag push. Do not edit by hand.
