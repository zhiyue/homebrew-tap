# zhiyue/homebrew-tap

Homebrew tap for tools maintained by [@zhiyue](https://github.com/zhiyue).

## Install

```bash
brew tap zhiyue/tap
```

### Formulas (CLI tools)

```bash
brew install zhiyue/tap/<formula-name>
```

### Casks (GUI apps)

```bash
brew install --cask zhiyue/tap/<cask-name>
```
